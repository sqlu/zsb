# ZSB
A discord selfbot written in TypeScript

## Installation

To install the dependencies, run:

```bash
npm install
or
pnpm install
or
yarn install

```

## Running the bot

To start the development environment, run:

```bash
npm run dev
or
pnpm dev
or
yarn dev
```

To build and run the bot, run:

```bash
npm run build && npm start
or
pnpm build && pnpm start
or
yarn build && yarn start
```
